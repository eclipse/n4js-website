Since the Eclipse N4JS project is in incubation phase, we do not provide release versions yet. You can find links to nightly builds (update site and products for Windows, Linux and macos) at the N4JS Eclipse project's download page.

Enfore AG originally introduced N4JS and hosts nightly builds that include Java 11. You can find links to nightly builds (update site and products for Windows, Linux and macos) at the Enfore N4JS download page.


Eclipse Installer (Oomph)

For developing extensions for the N4JS language (or simply to fix bugs ;-) ), the recommended way to install the Eclipse IDE and set up the N4JS Java sources in a developer workspace is to use the Eclipse Installer. This installer is available from Eclipse.org. Run the installer and follow these steps:

  1. Switch to Advanced Mode via the menu located in the upper-right corner
  2. Select a product, e.g. "Eclipse IDE for Eclipse Committers" with a product version (Neon is recommended)
  3. Double-click the entry Eclipse Projects/N4JS so that it is shown in the catalog view
  4. On the next page, configure your desired paths for the installation and workspace directories
  5. Begin installation
The installer will guide you through the rest of the installation process. The workspace and all plug-ins are downloaded and configured automatically. The necessary git repository is also fetched and configured in the workspace.

System Requirements: In all cases, Java 11 is required to be installed on your system. Node.js version 10.13+ is also required. The N4JS plugins require the Eclipse release 4.11.
